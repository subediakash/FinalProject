To compile this project , open android studio , build the project and run it on devices with android 6.0 or greater. (PaddleBall/app/src/main/java/com/example/paddleball) is where main files are located to build.

This was compiled on SDK 28, Minimum SDK for this app is 23 .

To play the game , tap right or left on the screen to move the paddle. You can also hold the tap for moving paddle faster.
